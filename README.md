# To-Do App for the Ensolvers Challenge

A simple To-Do application that let's you create, edit, complete and delete To-Do items.

## Technologies used

- Java 11
- Spring Boot 2.6.3
  - JPA
  - Spring Web
  - Lombok
- Maven 3.6.3
- Node.js 16.13.2
- React 17.0.2
- Axios 2.1.4
- MySQL 8.0.27
- SASS

## Requirements

To run this application, the following must be installed:

- Java JDK 11
- Node.js ^16.13
- MySQL 8
